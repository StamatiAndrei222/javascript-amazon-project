# JavaScript Amazon Project

## Descriere
Acest proiect reprezintă o aplicație web de tip e-commerce, inspirată de structura magazinului online Amazon. A fost realizat pentru a exersa dezvoltarea interfețelor web folosind HTML, CSS și JavaScript.

## Funcționalități principale
- afișarea produselor
- adăugarea produselor în coș
- actualizarea cantității produselor
- pagină de checkout
- pagină de comenzi
- pagină de tracking
- structurare a datelor și logicii în fișiere JavaScript separate

## Tehnologii utilizate
- HTML
- CSS
- JavaScript

## Scopul proiectului
Scopul acestui proiect a fost să îmi dezvolt abilitățile de lucru cu JavaScript, manipularea DOM-ului, organizarea codului pe module și construirea fluxului de bază al unei aplicații de tip magazin online.

## Ce am învățat
Prin realizarea acestui proiect, am exersat:
- manipularea elementelor din pagină cu JavaScript
- organizarea logicii aplicației în mai multe fișiere
- gestionarea coșului de cumpărături
- construirea unor pagini conectate între ele
- structurarea unui proiect frontend mai mare

## Rulare proiect
1. Descarcă proiectul sau clonează repository-ul.
2. Deschide fișierul `amazon.html` în browser.
3. Navighează prin paginile aplicației.

## Observație
Acest proiect este unul de practică și demonstrare a cunoștințelor de frontend, concentrat pe logica aplicației și interfața utilizatorului.
